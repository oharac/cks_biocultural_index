If the sample is representative of the population (of cultural keystone species globally) then this will give an estimate of uncertainty around the number of spp in each bin. However, we know this sample is problematic! So not sure if this is usefulâ€¦

For bootstrapping, sample with replacement for one iteration isnâ€™t any different than sample with replacement for two (or a thousand) iterations.

sims <- 1000
nspp <- nrow(bcv_raw)

set.seed(42)
bs_df <- data.frame(sim = rep(1:sims, each = nspp)) %>%
  mutate(bcv_code = sample(bcv_raw$bcv_code, size = n(), replace = TRUE)) %>%
  group_by(sim, bcv_code) %>%
  summarize(n_spp = n()) 
se_df <- bs_df %>%
  group_by(bcv_code) %>%
  summarize(n_mean = mean(n_spp),
            n_sdev = sd(n_spp),
            n_02.5 = quantile(n_spp, 0.025),
            n_97.5 = quantile(n_spp, 0.975),
            pct_mean = n_mean / nspp,
            pct_sdev = n_sdev / nspp,
            pct_02.5 = n_02.5 / nspp,
            pct_97.5 = n_97.5 / nspp)

ggplot(se_df, aes(x = bcv_code)) +
  geom_col(aes(y = n_mean, fill = bcv_code), color = 'grey40', show.legend = FALSE) +
  geom_point(data = sample_sum_df, 
             aes(y = n_spp), size = 3) +
  geom_errorbar(aes(ymin = n_mean - n_sdev, ymax = n_mean + n_sdev), 
                width = .2) +
  geom_errorbar(aes(ymin = n_02.5, ymax = n_97.5), color = 'red',

  scale_fill_manual(values = cat_clr$clr, breaks = cat_clr$bcv) +

Randomize IUCN and cultural vitality

What if IUCN status distribution is representative, and cultural vitality distribution is representative, but the two vars are not independent?
